# 引用元コピー機能

選択したテキストを学術的な引用形式で自動コピーするChrome拡張機能

## 機能

- 選択したテキストを右クリックで引用形式に変換
- Web記事と書籍の両方に対応
- 自動で著者名、URL、アクセス日付を取得
- クリップボードに自動コピー

## インストール方法

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 「デベロッパーモード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## 使い方

1. Webページ上でテキストを選択
2. 右クリックして「引用形式でコピー」を選択
3. 引用形式でクリップボードにコピーされます

## 出力例

```
"選挙のインターネット投票では、スマホによる手軽な投票の実現により若年層の投票率向上に繋がること..."1)

1) "ネット投票は実現するのか！？"．Business lesders square wisdom．2025．https://example.com，（参照 2025-12-08）．
```
