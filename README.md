# 引用元コピー機能

選択したテキストを学術的な引用形式で自動コピーするChrome拡張機能です。

## 機能

- 選択したテキストを右クリックで引用形式に変換
- Web記事と書籍の両方に対応
- 自動で著者名、URL、アクセス日付を取得
- クリップボードに自動コピー

## インストール方法

1. このリポジトリをクローンまたはダウンロード
2. Chrome で `chrome://extensions/` を開く
3. 「デベロッパーモード」を有効化
4. 「パッケージ化されていない拡張機能を読み込む」をクリック
5. ダウンロードしたフォルダを選択

## 使い方

1. Webページ上でテキストを選択
2. 右クリックして「引用形式でコピー」を選択
3. 引用形式でクリップボードにコピーされます

## 出力例

```
"デジタル技術の進化により、リモートワークの生産性が向上し、働き方の多様化が進んでいる。"1)

1) "新しい働き方の可能性"．ビジネストレンド．2024．https://example.com，（参照 2025-12-08）．
```
